ping 是基于 ICMP 协议工作的，所以要明白 ping 的工作，首先我们先来熟悉 ICMP 协议。

## ICMP 是什么？
ICMP 全称是 Internet Control Message Protocol，也就是互联网控制报文协议。

### ICMP 功能都有啥？
ICMP 主要的功能包括：

确认 IP 包是否成功送达目标地址、报告发送过程中 IP 包被废弃的原因和改善网络设置等。
